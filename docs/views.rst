=====
Views /polls/views.py
Шаблоны разметки /polls/templates/polls
=====

-----------

1. Views - обрабатывает сохранине информации в базу данных через модели (models) и вывод информации (templates)

2. index.html - вывод списка доступных вопросов, с возожностью проийти опрос (detail.html) и увидеть результаты (results.html)

3. detail.html - вывод списка вариантов ответов по вопросу, с возможностью выбора (голосования)

4. results.html - вывод результатов пройденных опросов пользователя с информацией о выбранных ответах
