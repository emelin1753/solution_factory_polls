=====
Модели базы данных
/polls/models.py
=====

-----------

1. Question - список вопросов. 
	Поля модели:
	question_text - текст вопроса
	question_type - тип вопроса (подразумеватеся текстовый ответ, выбор одного варианта или выбор из нескольких вариантов)
	pub_date - дата публикации, до даты публикации вопрос не доступен
	fin_date - дата окончания, после которой вопрос уже не доступен

2. Choice - варианты ответа. 
	Поля модели:
	question - ссылка по ключу на Question
	choice_text - текст варианта ответа

3. QuestionResults - хранилище выбранных ответов от пользователей. 
	Поля модели:
	question - ссылка по ключу на Question
	user - пользователь, в виде ссылкы по ключу на типовую модель django User. Может быть None для Anonymous
	votes_text - текст варианта/вариантов ответа пользователя
